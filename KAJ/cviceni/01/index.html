Javascript - DOM

  • Napište aplikaci, která data v proměnné todos objektu state vypíše do stránky
  • var state = {
	todos: [
		{ text: 'Hi', completed: true  },
		{ text: 'Hello', completed: false },
		{ text: 'Hi there!', completed: true },
	],
	filter: 'all'
};
  • Pro výpis dat do stránky použijde DOM metody jako document.createElement a Element.appendChild
  • Zkuste také vypsat data do stránký pomocí HTML stringu, zkuste document.innerHTML a Element.insertAdjacentHTML
